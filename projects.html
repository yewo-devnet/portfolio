Nthambi

Nthambi is a comprehensive web application designed to provide innovative solutions for network monitoring and management. It features an intuitive dashboard that displays real-time network metrics, allowing administrators to quickly identify and address potential issues.

The application includes automated alerts, in-depth analytics, and customizable reports to help network teams maintain optimal performance. One of its key strengths is the ability to visualize complex network topologies in an easy-to-understand format.

Network Monitoring Tool

This advanced network monitoring tool offers real-time monitoring of network performance, with automated alerts for potential issues and comprehensive analytics for network administrators. It uses SNMP to collect data from network devices and presents the information through an intuitive web interface.

The tool includes features such as bandwidth monitoring, device health tracking, and historical performance analysis. It's designed to be highly configurable, allowing administrators to set custom thresholds and notification rules based on their specific network requirements.

Python Django PostgreSQL SNMP Chart.js Celery Redis
Status: Ongoing Development
Started: September 2024

University Campus Network

I designed and implemented a scalable and secure network infrastructure for a university campus, supporting thousands of concurrent users across multiple buildings. The network architecture incorporates a hierarchical design with core, distribution, and access layers to ensure optimal performance and reliability.

Security measures include VLANs for traffic segmentation, access control lists, and a robust firewall implementation. The network also features redundancy at multiple levels to eliminate single points of failure, and supports diverse services such as VoIP, video conferencing, and high-speed internet access.

Cisco VLAN OSPF Firewall VPN QoS Network Security
Status: Completed
March - July 2024
